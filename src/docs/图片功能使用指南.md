# 图片功能使用指南

## 概述

BiSheng Latte 编辑器支持在 Markdown 笔记中插入、预览和同步图片。图片以 `![alt](bisheng://img/{uuid})` 格式嵌入文档，支持跨设备自动同步。

## 插入图片

### 粘贴插入

使用 **Ctrl + V** 直接粘贴剪贴板中的图片：

1. 截图或复制一张图片到剪贴板
2. 在编辑器中定位光标到目标位置
3. 按 **Ctrl + V**

编辑器会自动完成以下操作：
- 生成 UUID 作为文件名（如 `a1b2c3d4-...png`）
- 将图片保存到应用目录下的 `images/` 文件夹
- 在光标位置插入 `![image](bisheng://img/{uuid})` 语法
- 立即渲染图片预览
- 后台自动上传到服务器

### 超过 10MB 的图片

粘贴时如果图片文件超过 10MB，编辑器会自动压缩（降低分辨率），避免存储和同步开销过大。

## 图片预览

### 显示规则

- 图片始终以内联预览的方式显示在文档中
- **小图**（宽度 ≤ 编辑器页面宽度）：按原始尺寸展示，不放大不缩小
- **大图**（宽度 > 编辑器页面宽度）：等比缩放至页面宽度
- 高度始终按宽度等比自动计算

### 支持的图片格式

| 来源 | 支持格式 |
|------|---------|
| 粘贴 | PNG（剪贴板图片统一保存为 PNG） |
| Markdown 语法 | 取决于 `bisheng://` 解析器，通常支持 PNG/JPG/GIF/BMP |

## 修改与删除图片

### 修改图片

图片不支持直接编辑 URL。如需替换图片：

1. **删除**旧的图片语法（选中整行或 `![...](...)` 文本后按 Delete）
2. **重新粘贴**新的图片（Ctrl + V）

### 删除图片

1. 选中包含图片语法的整行文本
2. 按 Delete 或 Backspace 删除

> **删除语义（设计如此）：**
>
> - 去掉 Markdown 中的图片引用后，**客户端不会**调用服务端 `DELETE /api/images/{id}`。
> - **本机** `images/` 下的文件也不会因此自动删除。
> - **服务端**依赖孤儿 GC：上传后超过 `ImageGc:OrphanGraceDays`（默认 **7** 天）仍不被任何笔记引用则软删除；软删除后再经 `ImageRetentionDays`（默认 **30** 天）硬删磁盘文件。
> - 因此从笔记中删图到云端文件真正消失会有宽限期；需要立即清盘时由运维处理，而非同步协议保证。

## 图片同步

### 上传

- 粘贴图片后，后台服务每 **10 秒**自动扫描未上传的图片，逐张上传到服务器
- 上传完成后标记为已同步，不会重复上传

### 下载

- 打开其他设备创建的笔记时，如果图片未同步到本地，编辑器会显示「图片未同步」占位符
- 系统会自动触发后台下载，下载完成后自动刷新显示
- 如果下载失败，可点击「重新加载」按钮手动重试

### 增量拉取

- 后台服务每 **60 秒**检查一次服务端是否有新增图片，自动下载到本地

## 常见问题

### 图片显示为占位符

| 占位符文字 | 原因 | 解决方法 |
|-----------|------|---------|
| 「加载中...」 | 网络图片正在下载 | 等待加载完成 |
| 「图片未同步」 | 本地无缓存，等待服务器下载 | 等待自动下载，或点击「重新加载」 |
| 「图片不可用」 | 服务端和本地均无此图片 | 检查网络连接，确认图片已上传 |
| 「图片不存在」 | 本地路径无效或文件已删除 | 删除后重新粘贴 |
| 「图片解析器未配置」 | 编辑器未注入 IImageResolver | 检查应用初始化配置 |

### 图片位置偏移或压住文字

这是已修复的问题。图片使用固定间隙法计算行间距，与文字保持一致的间距。如仍出现异常，请检查 `TextView.LineSpacing` 设置。

### 图片太大占满页面

大图会自动缩放至编辑器页面宽度。如果图片仍然过大，可以在外部工具中缩小图片尺寸后重新粘贴。
